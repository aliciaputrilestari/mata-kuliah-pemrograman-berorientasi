PENGERTIAN POLIMORFISME DAN ABSTAKSI

Polimorfisme: Entitas sama dengan perilaku berbeda sesuai konteks. Abstraksi: Sembunyikan detail tak perlu, tampilkan hanya hal esensial.

PEWARIS

Pewarisan adalah konsep Pemrograman Berorientasi Objek (OOP) yang memungkinkan kelas anak (subclass) untuk mengambil alih serta menggunakan atribut dan fungsi yang dimiliki oleh kelas induk (superclass). Pada bahasa Dart, hubungan pewarisan dibuat dengan menggunakan kata kunci  extends .

// Nama  : Alicia Putri Lestari
// Nim   : 251410011
// Kelas : SI2A

class Hewan {
  void bersuara() => print('...');
}

class Kucing extends Hewan {
  @override
  void bersuara() => print('Meong');
}

void main() {
  Kucing k = Kucing();
  k.bersuara();
}

PENGANTAR POLIMORFISME

Polimorfisme berarti "banyak bentuk"—konsep inti OOP yang membuat entitas seperti method/kelas memiliki perilaku berbeda sesuai objek/konteksnya. Kode jadi fleksibel, bisa digunakan untuk berbagai objek tanpa logika terpisah, dan bekerja bersama dengan pewarisan serta abstraksi.

POLIMORFISME DENGAN INHERITANCE

Variabel parent bisa simpan objek child berbeda.

// Nama  : Alicia Putri Lestari
// Nim   : 251410011
// Kelas : SI2A

class Bentuk {
  void gambar() => print('Menggambar bentuk');
}

class Lingkaran extends Bentuk {
  @override
  void gambar() => print('Menggambar lingkaran');
}

class Persegi extends Bentuk {
  @override
  void gambar() => print('Menggambar persegi');
}

void main() {
  Bentuk bentuk1 = Lingkaran(); // Polimorfisme
  Bentuk bentuk2 = Persegi();

  bentuk1.gambar(); // Menggambar lingkaran
  bentuk2.gambar(); // Menggambar persegi
}

MANFAAT POLIMORFISM

Kode dinamis, mudah diperluas, bisa digunakan kembali, dan mudah dikelola.

IS DAN AS OPERATOR

 is  mengecek apakah objek termasuk tipe tertentu;  as  melakukan konversi tipe objek agar bisa mengakses method tertentu dengan aman.

// Nama  : Alicia Putri Lestari
// Nim   : 251410011
// Kelas : SI2A

class Kendaraan {}

class Mobil extends Kendaraan {
  void klakson() => print('Tin!');
}

void main() {
  Kendaraan k = Mobil();

  if (k is Mobil) {
    (k as Mobil).klakson();
  }

  // Safe casting
  (k as Mobil?)?.klakson();
}










PRAKTIK 1: POLIMORFISME DENGAN LIST


List bertipe parent bisa menyimpan berbagai objek subclass. Saat diulang,
setiap objek akan menjalankan method sesuai implementasinya sendiri.




// Nama  : Alicia Putri Lestari
// Nim   : 251410011
// Kelas : SI2A

class Hewan {
  void bersuara() => print('Hewan bersuara');
}

class Kucing extends Hewan {
  @override
  void bersuara() => print('Meong');
}

class Anjing extends Hewan {
  @override
  void bersuara() => print('Guk guk');
}

class Burung extends Hewan {
  @override
  void bersuara() => print('Cuit cuit');
}

void main() {
  List daftarHewan = [
    Kucing(),
    Anjing(),
    Burung(),
  ];

  for (var h in daftarHewan) {
    h.bersuara();
  }
}

ABSTRACT CLASS


Tidak bisa dibuat objek langsung, punya method abstract dan biasa, jadi dasar kelas anak.




// Nama  : Alicia Putri Lestari
// Nim   : 251410011
// Kelas : SI2A

abstract class Kendaraan {
  String? merk;

  void start(); // abstract

  void stop() {
    print('Kendaraan berhenti');
  }
}

class Mobil extends Kendaraan {
  @override
  void start() {
    print('Mobil $merk menyala');
  }
}

void main() {
  Mobil m = Mobil();
  m.merk = "Toyota";
  m.start();
  m.stop();
}

class User {
  String _username;
  String _password;
  String _email;

  User(this._username, this._password, this._email);

  // Getter untuk semua field
  String get username => _username;
  String get email => _email;

  // Setter dengan validasi
  set username(String v) {
    if (v.length >= 5) {
      _username = v;
    } else {
      print('Username min 5 karakter');
    }
  }

  set password(String v) {
    if (v.length >= 8) {
      _password = v;
    } else {
      print('Password min 8 karakter');
    }
  }

  set email(String v) {
    if (v.contains('@')) {
      _email = v;
    } else {
      print('Format email salah (Harus @)');
    }
  }

  void info() => print('User: $_username, Pass: $_password, Email: $_email');
}

void main() {

  user.username = 'aliciaputrilestari';
  user.password = 'rahasia987';

11 - STUDI KASUS - SISTEM LOGIN


 Logika autentikasi dan prosedur perubahan password




// Nama   : Alicia Putri Lestari
// Nim    : 251410011
// Kelas  : SI2A

class User {
  String _username;
  String _password;

  User(this._username, this._password);
  String get username => _username;

  bool login(String username, String password) {
    return _username == username && _password == password;
  }

  void gantiPassword(String lama, String baru) {
    if (_password == lama) {
      if (baru.length >= 8) {
        _password = baru;
        print('Password berhasil diubah.');
      } else {
        print('Gagal: Password baru minimal 8 karakter.');
      }
    } else {
      print('Gagal: Password lama tidak sesuai.');
    }
  }
}

void main() {
  print('--- Inisialisasi User ---');
  final User user1 = User('admin_alicia', 'securePass987');
  
  print('--- Uji Coba Login ---');
  if (user1.login('admin_alicia', 'securePass987')) {
    print('Login Berhasil!');
  }

  print('\n--- Uji Coba Ganti Password ---');
  user1.gantiPassword('securePass987', 'newSecurePass123');
  user1.gantiPassword('wrongPass', 'anyPass'); // Gagal
}

class RekeningBank {
  String _nomorRekening;
  double _saldo;

  RekeningBank(this._nomorRekening, this._saldo);

  void deposit(double jml) {
    if (jml > 0) {
      _saldo += jml;
      print('Deposit Rp $jml Berhasil.');
    }
  }

  void tarik(double jml) {
    if (jml > 0 && _saldo >= jml) {
      _saldo -= jml;
      print('Tarik Rp $jml Berhasil.');
    } else {
      print('Gagal: Saldo tidak cukup atau input tidak valid.');
    }
  }

  void cekSaldo() {
    print('Rekening: $_nomorRekening | Saldo: Rp $_saldo');
  }
}

void main() {
  var bank = RekeningBank('2514-10002', 3000000);
  bank.cekSaldo();
  bank.tarik(600000);
  bank.cekSaldo();
}
