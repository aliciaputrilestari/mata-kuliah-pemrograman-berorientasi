04 - Object Instantiation

Setelah class dibuat, kita bisa membuat object (instance) dari class tersebut. 

void main() {
  Car myCar = Car();
  myCar.model = "Sedan";
  myCar.color = "Merah";

  myCar.start();
  myCar.stop();
}

05 - Constructor

Constructor adalah method khusus yang otomatis dipanggil saat object dibuat. 

class Car {
  String model;
  String color;

  Car(this.model, this.color);
}

06 - Practice: Class Mahasiswa

class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  Mahasiswa(this.nama, this.nim, this.ipk);

  void tampilkanData() {
    print("Nama: $nama | NIM: $nim | IPK: $ipk");
  }
}

07 - Encapsulation

class Person {
  String _nama;
  int umur;

  Person(this._nama, this.umur);

  String get nama => _nama;
  set nama(String value) {
    _nama = value;
  }
}

08 - This Keyword

class Product {
  String nama;

  Product(this.nama);
}

09 - Named Parameter

class Produk {
  final String id;
  final String nama;
  final double harga;

  Produk({
    required this.id,
    required this.nama,
    required this.harga,
  });
}

10 - Kesimpulan

OOP membantu pengelolaan kode dengan konsep class dan object. Constructor mempermudah inisialisasi data. Encapsulation menjaga keamanan data. Dart mendukung OOP secara penuh.

11 - Class Hewan

Berikut contoh class Hewan dengan atribut nama dan method bersuara. 

class Hewan {
  String nama;

  Hewan(this.nama);

  void bersuara() {
    print("$nama sedang bersuara!");
  }
}

void main() {
  Hewan kucing = Hewan("Kucing");
  kucing.bersuara();
}
