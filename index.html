01 - Pengertian OOP

OOP adalah paradigma pemrograman yang berfokus pada objek. Setiap objek memiliki data (state) dan perilaku (behavior).

02 - Konsep Dasar

Class adalah blueprint. Object adalah instance. Attribute adalah data. Method adalah fungsi dalam class.

03 - Contoh Class Car

class Car {
  String? model;
  String? color;

  void start() {
    print("Car started!");
  }

  void stop() {
    print("Car stopped!");
  }
}

04 - Object Instantiation

void main() {
  Car myCar = Car();
  myCar.model = "Sedan";
  myCar.color = "Merah";
  myCar.start();
}

05 - Constructor

class Car {
  String model;
  String color;

  Car(this.model, this.color);
}

06 - Class Mahasiswa

class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  Mahasiswa(this.nama, this.nim, this.ipk);
}

07 - Encapsulation

class Person {
  String _nama;

  Person(this._nama);

  String get nama => _nama;
  set nama(String value) => _nama = value;
}

08 - This Keyword

class Product {
  String nama;

  Product(this.nama);
}

09 - Named Parameter

class Produk {
  final String id;
  final String nama;

  Produk({
    required this.id,
    required this.nama,
  });
}

10 - Kesimpulan

OOP mempermudah pengelolaan kode dengan konsep class dan object. Constructor membantu inisialisasi data. Encapsulation menjaga keamanan data.

11 - Class Hewan

class Hewan {
  String nama;

  Hewan(this.nama);

  void bersuara() {
    print("$nama sedang bersuara!");
  }
}
