01 PENGERTIAN OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berfokus pada objek sebagai unit utama. Setiap objek memiliki state (data) dan behavior (perilaku).

02 KONSEP DASAR

  • Class: Cetakan untuk membuat objek.
  • Object: Instance dari class.
  • Attribute: Data dalam class.
  • Method: Fungsi dalam class.

03 CONTOH CLASS SEDERHANA (CAR)

Contoh class Car dengan attribute dan method:

class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }

  void stop() {
    print('Car stopped!');
  }
}
