01 - Pengertian OOP

OOP adalah paradigma pemrograman berbasis object.

02 - Class

Class adalah blueprint untuk membuat object.

03 - Object

Object adalah instance dari class.

04 - Attribute

Attribute adalah data yang dimiliki class.

05 - Method

Method adalah fungsi dalam class.

06 - Constructor

class Mobil{
  String nama;
  Mobil(this.nama);
}

07 - Encapsulation

class Person{
  String _nama;
  Person(this._nama);
}

08 - This Keyword

class Product{
  String nama;
  Product(this.nama);
}

09 - Named Parameter

class Produk{
  final String id;
  Produk({required this.id});
}

10 - Kesimpulan

OOP mempermudah pengelolaan program dengan konsep object.

11 - Class Hewan

class Hewan{
  String nama;
  Hewan(this.nama);

  void bersuara(){
    print("$nama bersuara");
  }
}
