01 - Pengertian OOP

OOP adalah paradigma pemrograman yang berfokus pada object sebagai dasar pembuatan program.

02 - Konsep Dasar

Class adalah blueprint. Object adalah instance. Attribute adalah data. Method adalah fungsi.

03 - Class Car

class Car {
  String? model;
  String? color;

  void start(){
    print("Car started!");
  }
}

04 - Object

void main(){
  Car myCar = Car();
  myCar.model = "Sedan";
  myCar.start();
}

05 - Constructor

class Car {
  String model;
  Car(this.model);
}

06 - Class Mahasiswa

class Mahasiswa{
  String nama;
  String nim;

  Mahasiswa(this.nama,this.nim);
}

07 - Encapsulation

class Person{
  String _nama;
  Person(this._nama);

  String get nama => _nama;
}

08 - This Keyword

class Product{
  String nama;
  Product(this.nama);
}

09 - Named Parameter

class Produk{
  final String id;
  Produk({required this.id});
}

10 - Kesimpulan

OOP mempermudah pengelolaan kode dengan konsep class dan object.

11 - Class Hewan

class Hewan{
  String nama;

  Hewan(this.nama);

  void bersuara(){
    print("$nama sedang bersuara!");
  }
}
