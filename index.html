Class & Object

Class adalah blueprint, Object adalah instance dari class.

class Mobil {
  String nama;

  Mobil(this.nama);

  void jalan(){
    print("$nama sedang berjalan");
  }
}

Encapsulation

class Person {
  String _nama;

  Person(this._nama);

  String getNama(){
    return _nama;
  }
}
