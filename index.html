01 - Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berfokus pada objek sebagai unit utama. Setiap objek memiliki state (data) dan behavior (perilaku).

02 - Konsep Dasar

Class adalah blueprint untuk membuat objek. Object adalah instance dari class. Attribute adalah data dalam class. Method adalah fungsi dalam class.

03 - Contoh Class Sederhana (Car)

class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }

  void stop() {
    print('Car stopped!');
  }
}
